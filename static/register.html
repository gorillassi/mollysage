Как это работает Argon2 • X25519 • AES-GCM
  • На сервер отправляется только хэш пароля + публичный ключ.
  • Приватный ключ шифруется с ключа, полученного из пароля через Argon2id.
  • Даже при компрометации базы данных приватный ключ остаётся недоступен без пароля.
